What is Modded Minecraft?

Modded Minecraft.
With updates taking some time to release, and the content of the update usually keeping in spirit with the rest of the game, people took up the task to add their own content to the game.

There are a variety of mod types, including but not limited to:

Content Mods, which usually feature tech trees that have you build up factories or laboratories and create powerful weaponry and handy gadgetry for personal use. Automation is a common theme, enabling the player to spend time doing other things rather than mining stone or chopping wood.

Dimension Mods, adding new lands to discover and explore. Most fleshed-out dimension mods generally have special structures to encourage exploration, or have you start from zero, using strange new tools to work your way back up to the top.

Clientside Mods, that provide features to smooth out the experience, such as waypoints and maps to allow you to find your way easily.

Vanilla+ Mods, created in order to complement vanilla gameplay, rather than completely overtake it. Mods like this usually introduce minor features, such as the ability to place blocks more easily, or by adding more ways to cook food without rendering basic food obsolete.



How do I install mods?

In order to play Modded, you'll want a mod-loader and either some mods or a prebuilt modpack. The best and safest. place to download mods is on CurseForge, the official site for modded minecraft.

There are a myriad of modloaders, with the main one being the CurseForge App.
Other, more lightweight modloaders exist, however, such as PolyMC, which generally are much more lightweight than the App itself.

Both modloaders will have a button to either create a new profile, which will require you to add in either the Forge modloader or Fabric modloader, or to import a modpack zip file, which should come with everything you need to start playing.



Whats the difference between Forge and Fabric?

Forge was created first, and Fabric was created much later. Both versions apply mods to minecraft differently, meaning that mods only created for one modloader are completely incompatible with the other.
Forge is generally seen as being able to do more, while being more bulky, while Fabric is easier to code for, and is more lightweight.

For the average player, this generally just means that Forge has more content-based mods, and Fabric is more suitable for Vanilla+ playthroughs.
